# MDNET: Multi-Scale Differential Transformer for Efficient Lightweight Image Classification
**This is an official implementation of MDNET**.
MDNET incorporates a Multi-scale Differential Transformer (MDIFFormer) architecture that integrates diverse receptive fields along the channel dimension, enabling parallel extraction of multi-scale information and facilitating understanding of the global context. A differential attention mechanism inspired by physics is introduced to mitigate attention noise from irrelevant contexts, enhancing model robustness. Furthermore, label smoothing is applied to the probability distribution, introducing uniform noise to reduce over-reliance on true labels.
